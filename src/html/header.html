• 1. Скачати шрифти проекту в папку src/fonts
  • 2. Скачати SVG іконки в папку src/svgicons
  • 3. Призначити змінні для кольорів проекту в файлі src/scss/_global.scss
  • 4. Призначити змінні для шрифтів проекту в файлі src/scss/_global.scss
  • 5. Призначити, якщо потрібно, додаткові змінні в файлі src/scss/_global.scss
  • 6. Якщо потрібно підключаємо бургер меню та основну меню
  • 7. Додаємо CSS властивості що стосуються усіх сторінок в файл src/scss/paige.scss
  • 8. Додаємо CSS властивості що стосуються індивідуального контенту сторінки в файл src/scss/content.scss
  • 9. Перезапустити GULP = npm run dev
  • 10. Створюйте html, scss, js файли та насолоджуйтесь процесом
