На цьому етапі у Вас має бути завантажений та інстальований GULP. В цій збірці вся розробка ведеться в теці 'src' а gulp автоматично створив теку dist з результатом проекту.

В робочих файлах Ви можете подивитсь яким чином потрібно підключати та де розташовувати різні частини проекту. За бажанням всі файли в теці 'src' можна видалити та створити нові, або відредагувати згідно вашого проекту.

Файли js бажано залишити, вони роблять корисну справу.

За інструкцією нижче надана деяка інформація для швидкої та комфортної роботи над проектом.

Користуйтесь gulp із задоволенням та працюйте натхненно та з комфортом!


  • 1. Скачати шрифти проекту в папку src/fonts
  • 2. Скачати SVG іконки в папку src/svgicons
  • 3. Призначити змінні для кольорів проекту в файлі src/scss/_global.scss
  • 4. Призначити змінні для шрифтів проекту в файлі src/scss/_global.scss
  • 5. Призначити, якщо потрібно, додаткові змінні в файлі src/scss/_global.scss
  • 6. Перезапустити GULP = npm run dev

    • Примітка для того щоб нові шрифти перезаписалися у проект - потрібно видалити файл src/scss/fonts.scss. Це зроблено для того щоб при необхідності можна було внести зміни в шрифти проекту в ручну і вони не перезаписувался автоматично.

  • 7. Можно працювати над проектом. Нижче декілька не обов'язкових підказок
  • 8. Якщо потрібно підключаємо бургер меню та основну меню
  • 9. Додаємо CSS властивості що стосуються усіх сторінок в файл src/scss/paige.scss
  • 10. Додаємо CSS властивості що стосуються індивідуального контенту сторінки в файл src/scss/content.scss
  • 11. Перезапустити GULP = npm run dev
  • 12. Створюйте html, scss, js файли та насолоджуйтесь процесом
